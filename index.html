Simulator Use Instructions

States
  • To create a new state
 Mouse Left Click anywhere on the board.
  • To update a state label
  • To create a Start state
  • To create an Accept state
  • To delete an existing state
 Mouse Right Click over the state.
  • To move a state
 Drag the state to move.
Transitions
  • To create an edge
 Step 1: Mouse Double Left Click on the source state.

Step 2: Mouse Double Left Click on the destination state.
  • To update an edge label
  • To delete an edge
 Mouse Right Click over the transition edge.
